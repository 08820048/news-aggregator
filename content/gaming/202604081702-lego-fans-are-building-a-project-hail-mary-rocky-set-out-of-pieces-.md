LEGO has released only one [Project Hail Mary set](https://www.ign.com/articles/we-build-the-lego-project-hail-mary-ship-set-11389) so far, but it's been surprisingly popular. If you're looking to build a little Rocky pal from the movie, however, you're going to have to get creative. Which is exactly what fans have started doing.

There are a couple of different Rocky builds floating around Rebrickable, but the most popular option right now uses pieces from a retired groot set. The new MOC ("My Own Creation") comes from user [The\_Astral\_J](https://rebrickable.com/users/The_Astral_J/mocs/) and uses 310 pieces from the 476-piece I am Groot set.

## LEGO Rocky Alternative Build

[](https://assets-prd.ignimgs.com/2026/04/08/rocky-build-1775665691556.png)

Source: [Rebrickable](https://rebrickable.com/mocs/MOC-257725/The_Astral_J/scary-space-monster/#photos)

The unofficial Rocky build, which the user has titled "Scary Space Monster" is actually the second attempt from The\_Astral\_J at creating a LEGO version of the Project Hail Mary fan-favorite character. It uses 310 pieces from the I am Groot set, which has already been retired and is 'Jazz hands capable' according to the Rebrickable page. You can [download the instructions](https://rebrickable.com/mocs/MOC-257725/The_Astral_J/scary-space-monster/#) for this alternative build for free, and that retired Groot set is actually still available at Amazon.

Retired

### LEGO Marvel I am Groot Set

The build features five articulated arms, a tool armband, a scanning device, and an astrophage containment unit. If you can manage to procure the retired set, this is likely the best version of a LEGO Rocky we'll see anytime soon.

### The LEGO Cute Bunny Version

[](https://assets-prd.ignimgs.com/2026/04/08/1000x705p-1775666584240.png)

Source: [Rebrickable](https://rebrickable.com/mocs/MOC-256214/The_Astral_J/rocky-space-friend/#photos)

In addition to the I am Groot version, The\_Astral\_J has also created a Rocky set out of the LEGO 3-in-1 Cute Bunny set. This particular set has not yet been retired and is available for under $20 at Amazon right now. It uses only 155 parts of that set and is also 'Jazz hands capable', though it is far less detailed than the larger version.

### LEGO Creator 3 in 1 Cute Bunny

You can download the [instructions for this set](https://rebrickable.com/mocs/MOC-256214/The_Astral_J/rocky-space-friend/#details) for $3 at Rebrickable right now. It's a great option if you happen to have receive this [Easter-themed set](https://www.ign.com/articles/lego-easter-basket-gift-ideas) recently and need a better use for it than just a cute bunny.
