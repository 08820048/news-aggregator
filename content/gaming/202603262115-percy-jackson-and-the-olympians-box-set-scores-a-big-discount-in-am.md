[Amazon's Spring Sale](https://www.ign.com/articles/amazon-spring-sale-2026-best-deals) is here, and perhaps one of the biggest surprises so far is a quite extensive '3 for 2' sale. If you're unfamiliar, this event allows you to score three items for the price of two, which can usually save you quite a bit of cash.

Of all the items available in this promotion, one of the best is assuredly the Percy Jackson and the Olympians Box Set, which contains five books. With the [TV show](https://www.ign.com/articles/percy-jackson-and-the-olympians-review-disney-plus) recently wrapping up its second season, there's never been a better time to catch up on the source material! Don't miss your chance to save big on the Percy Jackson and the Olympians Box Set during the Amazon Spring Sale.

### Percy Jackson and the Olympians 5 Book Paperback Boxed Set (w/poster)

This box set for [Percy Jackson and the Olympians](https://www.ign.com/tv/percy-jackson-and-the-olympians) contains five books, with each containing art that flows from cover to cover when set together. A poster is also included, allowing you to share your love for the series outside a bookshelf. If you've never read or watched Percy Jackson before, this is a great starting point. Books 6 and 7 are available to purchase separately, so you can head there if you end up enjoying this box set.

By far, the best part of this deal is the fact that you can mix and match books, movies, and vinyl in this sale. That means if you have a book you want to grab with [a couple of 4K Blu-rays](https://www.ign.com/articles/amazon-3-for-2-4k-blu-ray-deal-spring-sale-2026), you can get all three items for the price of two, despite products in different categories.
