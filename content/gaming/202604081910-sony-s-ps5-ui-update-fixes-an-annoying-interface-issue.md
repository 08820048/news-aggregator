## Sony's testing a long-overdue change to the PS5 UI

   ![A screenshot of the PlayStation 5 interface featuring Elden Ring Nightreign artwork](https://static0.polygonimages.com/wordpress/wp-content/uploads/2026/04/20260408135431.jpg?&fit=crop&w=1600&h=900)Image: Sony Interactive Entertainment via Polygon

[Sony Interactive Entertainment](https://www.polygon.com/playstation/) is testing some changes to the [PlayStation 5](https://www.polygon.com/reviews/21549230/ps5-review-playstation-5-sony-next-gen/) user interface, including one that will address a relatively minor complaint that has long bothered console owners — specifically me. The proposed change will affect the top menu bar of the PS5's UI, according to images of the update from users taking part in [Sony's beta testing program](https://www.polygon.com/22538297/ps5-system-software-beta-test-sign-up/).

Currently, the PS5 UI features two primary sections, Games and Media. The former includes icons for a user's most recently played games and a "Welcome" information tab, as well as icons for the PlayStation Store, PlayStation Plus, and Game Library. The latter is home to multimedia and streaming apps like Netflix, YouTube, Crunchyroll, Spotify, and Twitch. Users tab between the Games and Media sections using L1 and R1 on a DualSense controller.

According to [some PS5 users](https://www.reddit.com/r/playstation/comments/1se03qt/new_playstation_home_screen_ui/) who have received an updated version of the UI, Sony plans to move the PlayStation Store, PlayStation Plus, and Game Library icons above the individual game icons. That change would certainly make it easier for PS5 owners to quickly jump into the PlayStation Store or their personal game library.

The beta PS5 UI is a minor tweak, but a welcome one. Sony hasn't made major changes to [the main user interface since the launch of the PlayStation 5 in 2020](https://www.polygon.com/2020/10/15/21517430/ps5-interface-user-experience-ui/), and being able to utilize the L1/R1 buttons for more than just quick navigation between two sections would be a big deal.

SIE added the customizable Welcome hub to the PS5 UI back in 2024, which it described as "a dedicated space you can customize with widgets, allowing you to see information at a glance before you begin your gaming sessions." The Welcome hub replaced the system's Explore tab.

No word yet on when Sony plans on rolling out this small-but-overdue change to the PS5 UI, but if you're interested in living that beta-testing life, you can [learn more about it and register your interest at the official PlayStation website](https://www.playstation.com/en-us/beta-program-at-playstation/).
