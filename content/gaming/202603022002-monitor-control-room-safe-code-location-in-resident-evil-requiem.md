The **Monitor Control Room safe** in [_Resident Evil Requiem_](https://www.polygon.com/resident-evil-requiem/) contains very helpful items, but, like other safes in the game, you can only open it after you find the code. In this case, the members of the Ark staff were kind enough to write the code on a piece of paper and leave it somewhere you could find.

To help you open this safe and get your hands on the rewards inside it, we explain **where to find the Monitor Control Room safe code in** **_Resident Evil Requiem_**, and detail the safe code for those who want to open it immediately.

\[**Ed. note**: This guide contains minor spoilers for _Resident Evil Requiem_.\]

**Update (March 2):** The safe code for insanity difficulty has been added.

* * *

## Monitor Control Room safe code location in Resident Evil Requiem

    ![A montage using two Resident Evil Requiem screenshots showing the location of the Monitor Control Room safe](https://static0.polygonimages.com/wordpress/wp-content/uploads/2026/02/rerequiem_guide_safe_monitorcontrolroom_map.jpg?q=49&fit=crop&w=825&dpr=2)Graphic: Paulo Kawanishi/Polygon | Source images: Capcom via Polygon

You can access the Monitor Control Room shortly after you unlock the **Lab Entrance door**. From the Lab Entrance, enter the Lobby, and use the adjacent door to locate the safe.

    ![A montage using two Resident Evil Requiem screenshots showing the location of the documento containing the Monitor Control Room safe code](https://static0.polygonimages.com/wordpress/wp-content/uploads/2026/02/rerequiem_guide_safe_monitorcontrolroom_document.jpg?q=49&fit=crop&w=825&dpr=2)Graphic: Paulo Kawanishi/Polygon | Source images: Capcom via Polygon

To find the Monitor Control Room code, you must go to **Bioweapon Repository 05**, which is just down the stairs. Check your ammo and prepare a couple bottles of acid to face not only regular zombies but also lickers in this area. The document with the code (along with some scrap) can be found inside the **open container in the middle row on the right** **side of the room**.

Since there isn't an alternative route to reach the Monitor Control Room from the Bioweapon Repository 05, you will need to backtrack to the Lobby and, then, access the Monitor Control Room.

## What is the Monitor Control Room safe code in Resident Evil Requiem?

    ![A Resident Evil Requiem screenshot showing the rewards you receive by opening the safe in the Monitor Control Room](https://static0.polygonimages.com/wordpress/wp-content/uploads/2026/02/rerequiem_guide_safe_monitorcontrolroom_rewards.jpg?q=49&fit=crop&w=825&dpr=2)Image: Capcom via Polygon

Finding the Monitor Control Room safe code requires you to either dodge the relentless lickers' attacks or spend valuable resources to kill them. If you'd rather save your resources and not backtrack to the safe, we've got you covered. To open the Monitor Control Room safe: turn the dial **left to 50, turn right to 60, and turn left to 80**. If you're playing on **insanity mode**, you must turn the dial **right to 20, left to zero, then right to 70**.

Opening this safe rewards you with **one bottle of acid, one empty bottle, and one med injector**.

If you're trying to open the other safes in _Resident Evil Requiem_, check out our guide on where to find every [safe](https://www.polygon.com/resident-evil-requiem-re9-safe-codes-combination-list-all/) along with its code.
