If you make a mistake in an Instagram comment, you can now finally fix it without having to delete the comment and make it again. On Thursday, [Instagram announced](https://www.threads.com/@instagram/post/DW6tJmFlv5U?xmt=AQF04qzLxm8d7wWpmUb3w2ND-I7D-2aP2TLPFl-R1EIilg) that users can now edit their comments within 15 minutes of posting them, which should hopefully give you ample time to clean up any errors.

You can make as many edits as you like while that 15-minute period is in effect, Meta spokesperson Nicole Rechtszaid tells _The Verge_. If you want to edit one of your comments, just tap the “Edit” text that shows up after you add a comment to a post. (I’m seeing the feature after making a couple test comments on the iOS app.) Any edited comments will have an “Edited” message in gray next to them.

Image: Instagram

Editable comments may seem like a small feature. But given how huge Instagram is — [it surpassed 3 billion monthly active users last year](https://www.theverge.com/news/784784/instagram-3-billion-monthly-users-reels-dms-updates) — I’m hoping it leads to comment sections with fewer typos.

The feature has been in testing with some users [since March](https://www.threads.com/@mattnavarra/post/DV0gpwvinha/instagram-is-finally-letting-users-edit-comments-on-posts-edited-comments-show).
